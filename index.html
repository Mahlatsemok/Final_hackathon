Latest Projects

Weather App

This Weather App allows users to view current weather conditions and forecasts for their location or any city worldwide. By integrating a third-party weather API, this app fetches real-time weather data and displays it in an easy-to-read format. The app provides users with essential information such as temperature, humidity, wind speed, and a short weather description. Technically, the app is built using HTML, CSS, JavaScript and the OpenWeatherMap API. Challenge is to handle asynchronous data from the APi, and the solution is to use JavaScript's fetch method to make asynchronous API requests and implemented promises to handle the data as it arrives. Another challenge is to manage cross-browser compatibility and ensure a smooth user experience on both mobile and demobile and desktop, and the solution is to use responsive design techniques with CSS media queries to ensure the layout adapts well on different screen sizes.

World Clock App

This World Clock App allows users to view the current time in multiple cities around the world. Whether you need to know the time in New York, Tokyo, London, or any other city, this app lets users select cities and instantly see the current time in each location. It also displays a list of cities in a neat, user-friendly interface, making it easy for travelers, remote teams, or anyone with global connections to stay on top of different time zones. Technically the app is built using HTML, CSS, JavaScript, and a time zone API (like WorldTimeAPI) to fetch current time and timezone data for each city. The clock updates automatically to display the current time without needing to refresh the page. Search Functionality users can search for a specific city and add it to their list of displayed cities. Challenge is integrating time zone data and dealing with daylight saving time (DST) changes, and solution is used an API like TimeZoneDB or WorldTimeAPI that automatically accounts for daylight saving time, ensuring that the displayed times are always accurate. Another Challenge is ensuring performance and speed when displaying multiple cities at once, Solution is optimized API calls and used caching techniques to minimize the number of requests and reduce loading times for users.
